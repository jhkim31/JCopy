# JCopy
https://jcopy.net/home
# 1. 개요

파일 전송을 위한 웹 어플리케이션

# 2. 목적

-   두 기기간 텍스트 및 파일 전송을 위해 개발
-   AWS, DevOps, node, kafka 등 다양한 기술 스택 공부 목적

# 특징

1. 로그인 없는 자료 전송 가능
2. 실시간 자료 확인 가능
